アクセス・道案内

ACCESS

ご来店前に必ずご確認ください

和招縁では、塩町店と福中店の2店舗を運営しております。
店舗ごとに場所・メニュー・雰囲気が異なりますので、
ご予約やご来店の際は店舗名を必ずご確認ください。

和招縁 塩町店

〒670-0904 兵庫県姫路市塩町177 アールビル 1F

お車でお越しの方

姫路駅方面から（約10分）

  1. 姫路駅南口を出て、大手前通り（国道2号線）を西へ進みます
  2. 「塩町」交差点を右折します
  3. 約200m直進すると、左手にアールビルが見えます
  4. 当店は1階にございます

※駐車場について：店舗前に3台分の専用駐車場をご用意しております

店舗外観と駐車場の写真

公共交通機関をご利用の方

JR姫路駅から（徒歩約15分）

  1. 姫路駅中央改札を出て、南口へ向かいます
  2. 駅前大通りを南へ直進します
  3. 「塩町」バス停付近で左折します
  4. 約100m進むと右手にアールビルがございます

バスをご利用の場合

姫路駅南口バスターミナルから神姫バス「塩町」バス停下車、徒歩1分

最寄りバス停の写真

和招縁 福中店

〒670-0017 兵庫県姫路市福中町78

お車でお越しの方

姫路駅方面から（約8分）

  1. 姫路駅南口から国道2号線を東へ進みます
  2. 「福中町」交差点を左折します
  3. 約150m直進すると、右手に当店がございます
  4. 店舗横の看板が目印です

※駐車場について：店舗裏に5台分の専用駐車場がございます

店舗外観と看板の写真

公共交通機関をご利用の方

JR姫路駅から（徒歩約20分）

  1. 姫路駅中央改札を出て、東口へ向かいます
  2. 東口を出て、線路沿いの道を東へ進みます
  3. 「福中町」の信号を右折します
  4. 約200m直進すると左手に当店がございます

バスをご利用の場合

姫路駅北口バスターミナルから神姫バス「福中町」バス停下車、徒歩2分

最寄りバス停と目印となる建物の写真
